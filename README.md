# FindTheNumber

A simple Java GUI game where the user guesses a random number using JOptionPane dialogs.

## How to Play
1. Run the `FindTheNumber.java` file.
2. A dialog box will prompt you to enter a number between 0 and 9.
3. The game will give hints: "enter upper guess" or "enter lower guess" until you guess correctly.
4. When you guess the number, a message will show how many attempts it took.

## Technologies Used
- Java
- JOptionPane (GUI)
